=== ИНСТРУКЦИЯ ПО ИЗМЕНЕНИЮ САЙТА ===

1. ОСНОВНЫЕ НАСТРОЙКИ
- В разделе <header> найдите блок с классом "logo-text"
- Вместо "example" впишите название вашей мастерской
- В теге <p> ниже можно изменить подпись (например: "Ремонт гитар с 2010 года")

2. КОНТАКТНЫЕ ДАННЫЕ
Ищите комментарии типа:
<!--Здесь измените все...-->
или
<!--Здесь замените на ваши данные-->

Замените:
- Номер телефона (+7 (123) 456-78-90)
- Email (guitar-repair@example.com)
- Адрес (Краснодар, ул. Гагарина, 123)
- Часы работы (Пн-Пт: 10:00 - 20:00 и т.д.)

3. СОЦИАЛЬНЫЕ СЕТИ
В разделах с классом "social-links":
- Оставьте только нужные вам иконки
- В тегах <a href="#"> замените # на реальные ссылки
- Лишние строки можно удалять полностью

4. ФОТОГРАФИИ
- Замените картинки в папке assets:
  - crasher_guitar.jpg (гитара до ремонта)
  - fixed_guitar.jpg (гитара после)
  - rest_guitar.jpg (реставрация)
  - prof_guitar.jpg (настройка)
  - people1-3.jpg (фото клиентов)

5. ДОПОЛНИТЕЛЬНО
- В футере измените год и название:
  "&copy; 2025 Example" → "&copy; 2025 ВашеНазвание"
- Для favicon замените файл logo.svg в папке assets

=== ВАЖНЫЕ ПРИМЕЧАНИЯ ===
1. Не удаляйте кавычки " и < >
2. Не меняйте названия классов (class="...")
3. Для переноса строк используйте тег <br>
4. Сохраняйте файл в кодировке UTF-8
5. Тестируйте изменения локально перед загрузкой на хостинг

=== ЕСЛИ ВСЁ СЛОМАЛОСЬ ===
1. Сохраните копию оригинального файла
2. Можно вернуть всё "как было"
3. Для сложных правок лучше наймите верстальщика
